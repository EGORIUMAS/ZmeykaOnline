Современная многопользовательская игра "Змейка" с онлайн и оффлайн режимами!

## ✨ Особенности

### 🌐 Онлайн режим
- Неограниченное количество игроков в комнате
- Динамическое увеличение поля (каждые 8 игроков поле увеличивается в 2 раза)
- Создание и присоединение к комнатам по коду
- Сохранение ника и комнаты при перезагрузке
- Владелец комнаты управляет запуском раунда
- Таблица побед для всех игроков

### 🎮 Оффлайн режим
- До 8 игроков на одном устройстве (4 человека + 7 ботов)
- Умные боты с разными стратегиями
- Локальная игра без интернета

### 📱 Мобильная поддержка
- Два режима управления: свайпы и кнопки
- Настройка в меню
- Оптимизация для планшетов
- На компьютерах можно выбирать количество локальных игроков

### 🎨 Интерфейс
- Красивый UI в зелёно-фиолетовых тонах
- Модульная структура кода
- Плавные анимации и переходы
- Адаптивный дизайн

## 🚀 Быстрый старт

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Настройка (опционально)

Создайте файл `.env` для настройки WebSocket:

```bash
cp .env.example .env
```

Отредактируйте `.env`:
```env
WEBSOCKET_HOST=localhost
WEBSOCKET_PORT=8000
```

### Запуск сервера

```bash
python server.py
```

Игра будет доступна по адресу: `http://localhost:8000`

### Сборка standalone версии

Для тестирования без сервера:

```bash
python build_standalone.py
```

Откройте `.develop/index.html` в браузере.

## 📝 Режимы игры

### Классика
Стандартные правила змейки. Столкновение с хвостом приводит к проигрышу.

### Полупризрак
Игроки случайно становятся невидимыми на 5 секунд. Хвост призрака не имеет хитбокса.

### Семейка недопризраков
Все змейки могут становиться призраками. Хвост невидим, но сохраняет хитбокс.

### Полноценный призрак
Игроки-люди являются призраками с начала игры.

### Компашка призраков
Все змейки являются призраками. Самый сложный режим!

### Магический шутер
Зачарованные яблоки дают специальные эффекты: оружие, неуязвимость или диарею.

## 🛠️ Технологии

- **Backend**: Python, Flask, Flask-SocketIO, Eventlet
- **Frontend**: Vanilla JavaScript (модульная структура), HTML5 Canvas
- **Стиль**: CSS3 с градиентами и анимациями
- **Связь**: WebSocket (Socket.IO)
- **Хранилище**: LocalStorage для сохранения настроек

## 🌐 Онлайн функции

- Создание комнат с уникальными кодами
- Автоматическое переподключение при разрыве связи
- Синхронизация состояния игры в реальном времени
- Таблица побед с сохранением между сессиями
- Динамическое масштабирование поля
