Красивая многопользовательская игра "Змейка" с поддержкой до 8 игроков одновременно!

## ✨ Особенности

- 🎮 До 8 игроков одновременно (4 человека + 7 ботов)
- 🎨 Красивый UI в зелёно-фиолетовых тонах
- 🎯 Несколько режимов игры:
  - Классический режим
  - Режимы "Призрак" (4 варианта)
  - Магический шутер
- 🎛️ Удобные слайдеры для выбора количества игроков
- 📱 Поддержка мобильных устройств (свайпы)
- ⛶ Полноэкранный режим
- 🤖 Умные боты с разными стратегиями

## 🚀 Быстрый старт

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск сервера

```bash
python server.py
```

Игра будет доступна по адресу: http://localhost:8080

## 🎮 Управление

### Клавиатура
- **Игрок 1**: ЦФЫВ
- **Игрок 2**: НПРО
- **Игрок 3**: ЗДЖЭ
- **Игрок 4**: Стрелки (↑←↓→)

### Мобильные устройства
- Свайпы по экрану в нужном направлении

Игра использует красивую зелёно-фиолетовую цветовую палитру:
- 💜 Фиолетовый (#7C4DFF, #9C27B0)
- 💚 Зелёный (#00C853, #00FF7F)

## 📝 Режимы игры

### Классика
Стандартные правила змейки. Столкновение с хвостом приводит к проигрышу.

### Полупризрак
Игроки случайно становятся невидимыми на 5 секунд. Хвост призрака не имеет хитбокса.

### Семейка недопризраков
Все змейки могут становиться призраками. Хвост невидим, но сохраняет хитбокс.

### Полноценный призрак
Игроки-люди являются призраками с начала игры.

### Компашка призраков
Все змейки являются призраками. Самый сложный режим!

### Магический шутер
Зачарованные яблоки дают специальные эффекты: оружие, неуязвимость или диарею.

## 🛠️ Технологии

- **Frontend**: Vanilla JavaScript, HTML5 Canvas
- **Стиль**: CSS3 с градиентами и анимациями
